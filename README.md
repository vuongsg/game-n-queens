# N Queens
N queens puzzle. You can try with variant 4x4, 6x6, 8x8 and 10x10 boards.

## Official website
<a href="https://game-n-queens-with-vuong.netlify.app/" target="_blank">https://game-n-queens-with-vuong.netlify.app/</a>

## Installing
```
npm install
```
or
```
yarn install
```

## Running
```
npm start
```
or
```
yarn start
```

## Technologies
- ReactJS
- Redux
- TypeScript
